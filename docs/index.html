108bridges
A UI for exploring Ontario bridge data. Interfaces with an API written in python. The interactive map was built using Leaflet. Can be used to view individual bridge details, (like construction history and specifications), as well as plot a route between bridges in the worst condition. Project was originally developed in a week, for fun!

API runs on a Heroku free tier dynamo, so expect a 30s wait-time on first load.
tsktask
A sweet and simple daily planner app for managing and creating tasks. Built using a custom lightweight framework. Uses the Web Storage API. Custom components are built using fancy ES6 classes. Made for a friend :) happy birthday!

watchitiwithme
I wanted to watch movies with my friends during quarantine, so I made this. Uses sockets for the chat (with custom emojis!) and video synchronization. The nodejs server also has a puppeteer webscraper for getting video source links. Designed to work only for exposed direct mp4 links. Is also compatible with external .srt captions.

Server runs on a Heroku free tier dynamo, so expect a 30s wait-time on first load.

A nice tool for visualizing large volumes ranging from 1 to 1016 gallons. Uses threejs for rendering. Super lightweight and lowpoly, and comes with some predefined sizes as well.
